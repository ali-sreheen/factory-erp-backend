# Factory ERP - Inventory Management System

نظام إدارة المخزون للمصنع، تم بناؤه باستخدام FastAPI للواجهة الخلفية و Tailwind CSS للواجهة الأمامية.

## المتطلبات الأساسية
- Python 3.8+

## طريقة التشغيل

### 1. تثبيت المكتبات (Dependencies)
قم بفتح سطر الأوامر (Terminal) في مجلد المشروع ونفذ الأمر التالي:
```bash
pip install -r requirements.txt
```

### 2. تشغيل خادم الواجهة الخلفية (Backend)
لِتشغيل خادم FastAPI، نفذ الأمر التالي:
```bash
uvicorn main:app --reload
```
سيعمل الخادم على الرابط: `http://localhost:8000`
- يمكنك عرض واجهة الـ API وتجربتها عبر Swagger UI من الرابط: `http://localhost:8000/docs`

### 3. فتح الواجهة الأمامية (Frontend)
لا تحتاج الواجهة الأمامية إلى خادم خاص للتشغيل، يمكنك فقط فتح ملف `index.html` الموجود داخل مجلد `frontend` في أي متصفح ويب.

## المميزات
- إضافة عناصر جديدة למخزون مع تحديد التصنيف (ألواح صاج، إكسسوارات، كشفات طوب).
- تحديث كميات المخزون (زيادة أو نقصان).
- عرض قائمة بجميع العناصر مع إمكانية الفلترة حسب التصنيف.
- واجهة تدعم اللغة العربية (RTL) بتصميم عصري باستخدام Tailwind CSS.
